Configuration

To use these pages in Azure User Flow:

  1. Copy the URL of each page from your browser
  2. Go to Azure Portal ‚Üí Your User Flow ‚Üí Page layouts
  3. Enable "Use custom page content" for each page type
  4. Paste the corresponding URL
  5. Save and test!
